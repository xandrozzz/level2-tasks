Что выведет программа? Объяснить вывод программы.

package main

    import (
        "fmt"
    )
    
    func main() {
        a := [5]int{76, 77, 78, 79, 80}
        var b []int = a[1:4]
        fmt.Println(b)
    }

Ответ:

Программа выведет

    [77 78 79]

Слайс b создается на основе массива a, первый индекс включительный, последний нет.
В слайс b попадут элементы с индексами 1,2,3.
